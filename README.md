# AgentScanner

AgentScanner 是一个智能体安全网关，用于监控和管理智能体的安全访问。它提供了智能体管理、策略管理、日志管理等核心功能，帮助用户确保智能体的安全运行。

## 文档目录

- [服务部署手册](docs/deployment.md)：包含环境要求、后端部署、前端部署、Nginx配置和启动顺序
- [技术文档](docs/technical.md)：包含项目架构、技术栈和核心功能说明

├── backend/          # 后端代码
├── frontend/         # 前端代码
